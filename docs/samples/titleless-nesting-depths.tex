\newcommand{\caselead}[1]{%
  \clearpage
  {\Large\bfseries #1\par}
  \medskip
  Every box in this case is a breakable tcolorbox without a title option.
  The visible labels below are ordinary body text, not tcolorbox titles.
  The page breaks are caused by the amount of prose, not by explicit breaks
  inside the boxes.
  \medskip
}

\newcommand{\innertext}{%
  \lipsum[1-5]
  \medskip
  \lipsum[6-9]
}

titleless nested breakable boxes do not acquire unwanted vertical title
clearance when the nesting depth changes.

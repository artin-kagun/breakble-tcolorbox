\newcommand{\continuousproofparagraph}[1]{%
  #1回目の確認として、$V$ の基底
  \[
    u_1,\dots,u_k,v_1,\dots,v_r
  \]
  に関する表示を比べる。任意の $x\in V$ は
  \[
    x=b_1u_1+\cdots+b_ku_k+a_1v_1+\cdots+a_rv_r
  \]
  と一意的に書ける。ここで $f(x)=0$ ならば、像に残る成分の係数は
  $a_1=\cdots=a_r=0$ でなければならない。したがって、核に落ちる部分と
  像に残る部分は、同じ基底表示の中で分離して扱える。

}

\section{一本の内側 SubProof がページをまたぐ検証}

このPDFでは、内側の \texttt{SubProof} を途中で閉じたり、新しい
\texttt{SubProof} を開始したりしない。一本の入れ子ボックスがそのまま
ページをまたぐとき、左側の破線が不自然に重なったり途切れたりしないことを確認する。

\begin{Proof}{連続する内側証明}
外側の証明を少し進めてから、長い内側証明に入る。

\begin{SubProof}[ページをまたぐ一本の確認]
\continuousproofparagraph{1}
\continuousproofparagraph{2}
\continuousproofparagraph{3}
\continuousproofparagraph{4}
\continuousproofparagraph{5}
\continuousproofparagraph{6}
\continuousproofparagraph{7}
\continuousproofparagraph{8}
\continuousproofparagraph{9}
\continuousproofparagraph{10}
\continuousproofparagraph{11}
\continuousproofparagraph{12}
\continuousproofparagraph{13}
\continuousproofparagraph{14}
\continuousproofparagraph{15}
\continuousproofparagraph{16}
\continuousproofparagraph{17}
\continuousproofparagraph{18}
\end{SubProof}

内側証明の後に、外側の証明へ戻る。この行は、入れ子ボックスの終了後に
外側のボックスが自然に続くことを確認するために置いている。
\end{Proof}
